# Mentora

Mentora is a web-based mentorship platform where mentors and students connect through structured, task-based courses across any field, from academics to hobbies. Mentors create task-driven courses and provide direct feedback, while students track their progress visually as they complete tasks. It supports both one-to-one and one-to-many learning, offering a more engaging and accountable alternative to passive video-lecture platforms.
